# VRT baseline snapshots

PNG baselines are generated by Playwright next to this folder (see `snapshotDir` in `playwright.config.ts`). Expected layout:

`ui-playground.spec.ts-snapshots/*.png` (names may include `-chromium-linux` depending on Playwright version).

## Generate / update baselines

From repository root (after `pnpm install` and `pnpm exec playwright install chromium` inside `apps/web`):

```bash
pnpm --dir apps/web run build
pnpm --dir apps/web exec playwright test --update-snapshots
```

Commit the updated files under `tests/visual/__screenshots__/`.

**Tip:** Generate baselines on **Linux** (or in CI with `ubuntu-latest`) so pixel output matches the GitHub Actions runner.
